# Design Context — CPSC Recall Violation Monitoring System

## Target Audience
Simulated CPSC (Consumer Product Safety Commission) employees — government regulatory workers managing product recalls, violation investigations, and adjudications.

## Use Cases
1. CPSC Managers: prioritize recalls, assign investigators
2. Investigators: monitor assigned violations, annotate marketplace listings
3. Sellers: view violations against them, submit responses
4. Admins: manage user roles, batch import recall data

## Brand Personality / Tone
**Professional SaaS** — think Linear or Notion meets government authority. Trustworthy, dense, efficient. Not flashy. Not sterile. Competent.

## Color System
- **Primary**: `#0D47A1` — authoritative government blue (Material Design blue 900)
- **Secondary**: `#E65100` — alert amber/orange for warnings
- **Background**: White for cards/paper, light blue-gray (`#ECEFF1`) for page
- **Font**: Source Sans 3 (government-adjacent, professional sans-serif)

## Anti-Pattern Flags
- No glassmorphism or glow effects
- No gradient text
- No pure decorative animations
- Dense information layouts are correct for this domain — don't "simplify" into empty space
- Role badges (Admin=red, Manager=blue, Investigator=green, Seller=gray) are intentional
